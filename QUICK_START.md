# Quick Start Guide

## To Start the Development Server

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Project Overview

This is a React TypeScript application built with create-react-app that displays a hierarchical tree view of vessel equipment.

### Key Features Implemented

1. ✅ **Hierarchical Tree View** - Recursive tree structure
2. ✅ **Expand/Collapse** - Interactive nodes with + and - buttons
3. ✅ **Node Type Styling** - 4 different node types with distinct colors
4. ✅ **Search Functionality** - Filter nodes with highlight matching
5. ✅ **Zoom Controls** - Zoom in, out, and reset
6. ✅ **TypeScript** - Full type safety
7. ✅ **Clean Architecture** - Well-organized component structure

### Node Types

- **Equipment Type** (Light Blue)
- **Equipment** (Dark Blue)
- **Assembly** (Light Red/Pink)
- **Component** (Light Green)

### Usage

1. Click the `+` button on any node to expand it
2. Click the `−` button to collapse
3. Type in the search box to filter nodes
4. Use zoom controls to adjust view size
